# FocusFlow — نسخه‌ی ویندوز

این یه برنامه‌ی پایتون تک‌فایلیه که دقیقاً مثل نسخه‌ی وب و اندروید رفتار می‌کنه:
- همون برنامه‌ی زمانی (بلوک‌های ۱۰۵ دقیقه‌ای مطالعه + ۱۵ دقیقه استراحت ادغام‌شده، ناهار ۱۳:۰۰، شام ۲۱:۰۰)
- وقتی زمان آلارم برسه، یه پنجره‌ی **تمام‌صفحه، همیشه بالا** باز می‌شه که نه با دکمه‌ی X، نه با Alt+F4 بسته نمی‌شه
- تنها راه بستنش حل کردن یکی از همون ۱۰۰ سؤال ریاضی بانک اصلیه
- فعال/غیرفعال کردن هر session، تغییر اسم بخش‌های مطالعه (کلیک روی اسم)، و ریست کردن — دقیقاً مثل نسخه‌های قبلی

## چطور تبدیلش کنم به یه فایل .exe (فقط یک‌بار، ۲ دقیقه)

نیازی به تجربه‌ی برنامه‌نویسی نداری، فقط این ۴ تا دستور رو دنبال کن:

### قدم ۱: نصب پایتون (اگه از قبل نداری)
برو **python.org/downloads** و نسخه‌ی ویندوز رو دانلود و نصب کن.
⚠️ موقع نصب، حتماً تیک گزینه‌ی **"Add Python to PATH"** رو بزن (پایین همون صفحه‌ی نصب).

### قدم ۲: باز کردن ترمینال (Command Prompt / PowerShell)
- دکمه‌ی ویندوز رو بزن، تایپ کن `cmd`، اینتر بزن.

### قدم ۳: نصب PyInstaller (ابزار تبدیل به exe)
توی همون پنجره‌ی سیاه، این رو تایپ کن و اینتر بزن:
```
pip install pyinstaller
```
صبر کن تموم بشه (کمتر از یک دقیقه).

### قدم ۴: ساخت exe
اول برو به پوشه‌ای که فایل `focusflow.py` توشه. مثلاً اگه توی Downloads هست:
```
cd Downloads\FocusFlowWindows
```
بعد این دستور رو بزن:
```
pyinstaller --onefile --noconsole --name FocusFlow focusflow.py
```
صبر کن (حدود ۱-۲ دقیقه). وقتی تموم شد، یه پوشه‌ی جدید به اسم `dist` می‌سازه.

### قدم ۵: اجرا
داخل پوشه‌ی `dist`، یه فایل هست به اسم **FocusFlow.exe** — همینه! دابل‌کلیک کن، اجرا می‌شه.
می‌تونی این فایل رو هر جا بخوای کپی کنی (روی Desktop مثلاً) — دیگه نیازی به پایتون یا هیچ چیز دیگه نداره، خودش مستقله.

## راه‌اندازی خودکار موقع روشن شدن ویندوز (اختیاری)
اگه می‌خوای هر بار که ویندوز روشن می‌شه، اپ خودش اجرا بشه:
۱. کلید ویندوز + R رو بزن، تایپ کن `shell:startup`، اینتر
۲. یه پوشه باز می‌شه — یه **shortcut** از `FocusFlow.exe` بساز و بذارش همون‌جا (راست‌کلیک روی exe → Create shortcut، بعد shortcut رو بکش تو اون پوشه)

## نکات
- تنظیمات (session های فعال/غیرفعال، اسم‌های تغییریافته) توی
  `%APPDATA%\FocusFlow\focusflow_data.json` ذخیره می‌شه.
- تا وقتی برنامه بازه (حتی minimize شده)، آلارم‌ها کار می‌کنن. اگه کامل ببندیش
  (از پنجره‌ی تأیید خروج)، آلارم‌ها متوقف می‌شن — پس بهتره بازش بذاری یا با
  استارتاپ خودکار (بالا) اجرا بشه.
- اگه خواستی زمان‌بندی رو عوض کنی (ساعت ناهار/شام، طول بلوک مطالعه)، فایل
  `focusflow.py` رو با هر ادیتور متنی (مثل Notepad) باز کن و تابع
  `generate_schedule()` رو ویرایش کن، بعد دوباره قدم ۴ رو تکرار کن.
